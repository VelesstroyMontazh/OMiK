# Как запустить программу (простыми словами)

Папка проекта: **`C:\Otchet_OP_Marina\OMiK_VSM`**

---

## Шаг 1. Один раз установить (если ещё не делали)

**Проще:** дважды щёлкните **`INSTALL.bat`** в папке проекта (всё установит сам).  
(Файл **`УСТАНОВКА.bat`** — то же самое, вызывает `INSTALL.bat`.)

Или вручную в **PowerShell**:

```text
bun install
pip install -r mini-services\excel-service\requirements.txt
```

Создать базу для списка файлов (один раз):

```text
.zscripts\setup-db.cmd
```

(Нужны **Bun** или Node, и **Python 3**.)

**Запуск с фоновой очередью in-process (рекомендуется, без Redis):** просто **`START.bat`**.

**Celery + Redis** — только опционально, см. **[ЗАПУСК-Celery.md](ЗАПУСК-Celery.md)** (без Docker).

---

## Шаг 2. Запуск программы (каждый раз)

### Способ А — проще всего

1. В проводнике откройте папку **`C:\Otchet_OP_Marina\OMiK_VSM`**
2. **Дважды щёлкните** файл **`START.bat`** (или **`ЗАПУСК.bat`** — то же самое)
3. Появится чёрное окно — **не закрывайте его**, пока работаете
4. Подождите 10–20 секунд, пока появится строка вроде `Ready` / `compiled`

### Способ Б — из Cursor / VS Code

1. Меню **Terminal → New Terminal**
2. Команда:

```powershell
.\.zscripts\start-dev.ps1
```

---

## Шаг 3. Открыть в браузере

Скопируйте в адресную строку браузера (Chrome, Edge):

**http://127.0.0.1:3000**

Должен открыться **главный экран** с карточками модулей.

---

## Шаг 4. Куда нажимать в программе

### Модуль «Затраты по билетам»

1. На главном экране нажмите карточку **«Затраты по билетам»** (фиолетовая, с иконкой билета).
2. Сверху вкладки:
   - **Дашборд** — сводка и графики
   - **ВелесстройМонтаж** — данные первой организации
   - **Стройконстракшен** — данные второй организации
3. Выберите **ВелесстройМонтаж** (или СК), затем подвкладку:
   - **Загрузить и обработать** — Excel-файлы
   - **Таблица данных** — результат A–U

### Загрузка билетов (по шагам)

На вкладке **«Загрузить и обработать»**:

| Шаг | Что нажать |
|-----|------------|
| 1 | **Выбрать файлы** или перетащить Excel в зону загрузки |
| 2 | **Загрузить в ВелесстройМонтаж** (или «Обработать и отобразить» — сделает загрузку и обработку сразу) |
| 3 | Дождаться **зелёного статус-бара** вверху (может быть несколько минут) |
| 4 | Справа появятся **«Исходные файлы в программе»** — не 0 |
| 5 | **Обработать и отобразить** — если ещё не нажимали на шаге 2 |
| 6 | Откроется **Таблица данных** с результатом |

> **Важно:** файлы в «Очереди» — это ещё не база. Пока не нажали «Загрузить» или «Обработать», будет ошибка «Нет сырых данных».

## Как понять, что всё работает

| Проверка | Адрес |
|----------|--------|
| Сайт | http://127.0.0.1:3000 |
| Сервер Excel (фон) | http://127.0.0.1:3031/api/health — должно быть `{"status":"ok"}` |

Проверка портов (PowerShell):

```powershell
.\.zscripts\check-ports.ps1
```

---

## Если что-то не открывается

| Проблема | Что сделать |
|----------|-------------|
| Страница не грузится | Окно `ЗАПУСК.bat` закрыто? Запустите снова |
| «Backend unavailable» / health 8 с | Закройте окно, снова **START.bat**; проверьте http://127.0.0.1:3031/api/health |
| `DATABASE_URL` not found | Запустите **`.zscripts\setup-db.cmd`**, перезапустите **START.bat** |
